Why AMD?

Differentiation of matrix functions is still not well understood. There is no "simple calculus" for computing matrix derivatives and therefore calculating matrix derivatives can be very time and memory consuming. We want to build on known results to define a matrix-derivative calculus.

Objective

The objective of AMD is to make matrix derivatives simple for both humans and computers. We want symbolic differentiation to be simple and optimizing matrix-valued functions to be efficient in computation and storage. AMD is an open-source C++ library to accomplish this.

Benefits

AMD can be used to optimize scalar-matrix functions. Additionally, AMD can be used for:
  • Optimizing proximal gradient methods
  • Taylor expansions for sensitivity analysis
  • And more!
